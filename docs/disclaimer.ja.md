# 免責事項

本ソフトウェア（jquants-mcp）は [J-Quants API v2](https://jpx-jquants.com/) から
取得した日本株データを Claude 等の MCP クライアントで利用するための技術ツールです。
投資判断のための参考情報を提供するものであり、投資関連の活動に利用する前に以下の
点を必ずお読みください。

## 投資助言ではありません

本ソフトウェアおよびその出力は、**投資助言・勧誘・推奨を行うものではありません**。
データアクセス・分析ツールであり、金融助言サービスではありません。

## データの正確性は保証しません

表示される情報の **正確性・完全性・適時性** について保証するものではありません。
データは J-Quants から取得しており、誤り・遅延・欠落が含まれる可能性があります。
重要な数値は必ず公式の開示資料（決算短信、有価証券報告書など）でご確認ください。

## 投資は自己責任で

**投資判断は利用者ご自身のリスクと責任** で行ってください。株価は変動し、損失が発生
する可能性があります。本ソフトウェアの各ツール（チャート、スクリーナー、財務情報
サマリー）が示す過去の実績は、将来の運用成果を保証するものではありません。

## 業登録を受けていません

本ソフトウェアの作者は、日本の **金融商品取引法に基づく金融商品取引業者の登録を
受けていません**。他の法域においても投資顧問業の認可を受けていません。

## J-Quants の利用規約

本ソフトウェアを通じて取得したデータの利用は、[J-Quants の利用規約](https://jpx-jquants.com/)
に従います。利用者は自身の J-Quants 契約プランの制限およびデータ再配布ポリシーを
遵守する責任があります。

データの不正確性、サービス中断を含むがこれに限らない）について、作者は **一切の
